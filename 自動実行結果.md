2. 「Repositories」タブをクリック
3. 「New」ボタンをクリック
4. リポジトリ名: `faq-knowledge-base`
5. 「Create repository」をクリック

### 2. GitHubにプッシュ

リポジトリを作成したら、ターミナルで以下を実行：

```bash
cd /Users/nakajima-yuusuke/Desktop/Cursor/faq-knowledge-base
git push -u origin main
```

**認証が必要な場合:**
- GitHubのユーザー名とパスワード（またはPersonal Access Token）の入力が求められます
- パスワードの代わりにPersonal Access Tokenを使うことをおすすめします
- トークンの作成: https://github.com/settings/tokens

### 3. Renderでデプロイ

プッシュが完了したら：

1. https://render.com にアクセス
2. 「Get Started for Free」→「Continue with GitHub」
3. 「New +」→「Web Service」
4. リポジトリ `faq-knowledge-base` を選択
5. 設定を入力:
   - **Build Command**: `npm install && cd client && npm install && npm run build`
   - **Start Command**: `node server/index.js`
   - **Environment Variables**: `NODE_ENV` = `production`
6. 「Create Web Service」をクリック
