## 次のステップ（簡単3ステップ）

### ステップ1: GitHubでリポジトリを作成（2分）

1. **https://github.com にアクセスしてログイン**
2. **右上の「+」→「New repository」をクリック**
3. **リポジトリ名を入力**: `faq-knowledge-base`
4. **「Create repository」をクリック**

### ステップ2: GitHubにコードをプッシュ（1分）

ターミナルで以下を実行してください：

```bash
cd /Users/nakajima-yuusuke/Desktop/Cursor/faq-knowledge-base

# [あなたのGitHubユーザー名]を実際の値に置き換える
git remote add origin https://github.com/[あなたのGitHubユーザー名]/faq-knowledge-base.git
git branch -M main
git push -u origin main
```

**GitHubユーザー名がわからない場合:**
- GitHubの右上のアイコンをクリック
- ユーザー名が表示されます

### ステップ3: Renderで公開（5分）

1. **https://render.com にアクセス**
2. **「Get Started for Free」→「Continue with GitHub」をクリック**
3. **「New +」→「Web Service」をクリック**
4. **リポジトリを選択**: `faq-knowledge-base`
5. **設定を入力:**
   - **Name**: `faq-knowledge-base`
   - **Build Command**: 
     ```
     npm install && cd client && npm install && npm run build
     ```
   - **Start Command**: 
     ```
     node server/index.js
     ```
   - **Environment Variables**: 
     - Key: `NODE_ENV`
     - Value: `production`
6. **「Create Web Service」をクリック**
7. **5-10分待つ**
8. **完了！URLが表示されます**
