# ウェブ上に公開する方法（超シンプル）

## 結論：Renderを使えば5分で公開できます

Renderは無料でウェブ上に公開できるサービスです。GitHubアカウントがあれば、すぐに公開できます。

---

## 手順（たった3ステップ）

### 1. GitHubにコードをアップロード（2分）

**ターミナルで以下を実行：**

```bash
cd /Users/nakajima-yuusuke/Desktop/Cursor/faq-knowledge-base

# Gitを初期化
git init
git add .
git commit -m "Initial commit"

# GitHubでリポジトリを作成してから、以下を実行
# （GitHubのユーザー名とリポジトリ名を入力）
git remote add origin https://github.com/[ユーザー名]/[リポジトリ名].git
git branch -M main
git push -u origin main
```

**GitHubでリポジトリを作る方法：**
1. https://github.com にログイン
2. 右上の「+」→「New repository」
3. 名前を入力して「Create repository」

---

### 2. Renderで公開（3分）

1. **https://render.com にアクセス**
2. **「Get Started for Free」→「Continue with GitHub」でログイン**
3. **「New +」→「Web Service」をクリック**
4. **GitHubリポジトリを選択**
5. **以下を入力：**
   - **Build Command**: `npm install && cd client && npm install && npm run build`
   - **Start Command**: `node server/index.js`
   - **Environment Variables**: `NODE_ENV` = `production`
6. **「Create Web Service」をクリック**
7. **5-10分待つ**

---

### 3. 完了！

表示されたURL（例: `https://faq-knowledge-base.onrender.com`）が、あなたのアプリの公開URLです。

このURLを外注者に共有するだけで、世界中のどこからでもアクセスできます。

## よくある質問

**Q: 無料で使えるの？**
A: はい、無料プランがあります。15分間アクセスがないとスリープしますが、最初のアクセス時に自動で起動します。

**Q: データは消えないの？**
A: はい、データは永続化されます。サービスを削除しない限り、データは残ります。

**Q: もっと簡単な方法はないの？**
A: Renderが最も簡単です。GitHubアカウントがあれば、数クリックで公開できます。

**Q: カスタムドメインは使える？**
A: はい、有料プラン（$7/月）でカスタムドメインが使えます。
