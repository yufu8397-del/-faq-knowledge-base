# GitHubにプッシュする手順

## リポジトリ作成完了 ✅

次は、コードをGitHubにアップロード（プッシュ）します。

## 手順

### ターミナルで以下を実行：

```bash
cd /Users/nakajima-yuusuke/Desktop/Cursor/faq-knowledge-base
git push -u origin main
```

### 認証情報の入力

実行すると、認証情報の入力が求められます：

1. **Username（ユーザー名）**: 
   ```
   yufu8397-del
   ```
   と入力してEnter

2. **Password（パスワード）**: 
   - GitHubのパスワードを入力
   - または、Personal Access Tokenを使用（推奨）

---

## Personal Access Tokenを使う方法（推奨）

GitHubのパスワードではなく、Personal Access Tokenを使うことをおすすめします。

### トークンの作成方法：

1. **https://github.com/settings/tokens にアクセス**
2. **「Generate new token (classic)」をクリック**
3. **Note（メモ）**: `faq-deploy` など適当な名前
4. **Expiration（有効期限）**: 適当な期間を選択（例: 90 days）
5. **Select scopes（スコープ）**: 
   - ✅ **`repo`** にチェック（すべてのリポジトリへのアクセス）
6. **「Generate token」をクリック**
7. **表示されたトークンをコピー**（この画面を閉じると二度と見れません！）

### プッシュ時に使用：

```bash
git push -u origin main
```

- **Username**: `yufu8397-del`
- **Password**: [コピーしたトークンを貼り付け]

## 次のステップ：Renderでデプロイ

プッシュが完了したら、Renderでデプロイします：

1. **https://render.com にアクセス**
2. **「Get Started for Free」→「Continue with GitHub」**
3. **「New +」→「Web Service」をクリック**
4. **リポジトリ「faq-knowledge-base」を選択**
5. **設定を入力：**
   - **Build Command**: `npm install && cd client && npm install && npm run build`
   - **Start Command**: `node server/index.js`
   - **Environment Variables**: `NODE_ENV` = `production`
6. **「Create Web Service」をクリック**
7. **5-10分待つ**

完了！
