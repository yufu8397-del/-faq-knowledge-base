# 次のステップ：Renderで公開

## 現在の状況
✅ GitHubでRenderアプリをインストールしようとしている

## 次の手順

### 1. GitHubでRenderアプリをインストール
現在の画面で：
- 「すべてのリポジトリ」または「選択したリポジトリのみ」を選択
- 「インストール」ボタンをクリック

### 2. Renderのダッシュボードに移動
1. https://render.com にアクセス
2. ログイン（GitHubアカウントで）

### 3. 新しいWebサービスを作成
1. ダッシュボードで「New +」ボタンをクリック
2. 「Web Service」を選択
3. 「Connect account」でGitHubアカウントを接続（まだの場合）
4. リポジトリを選択（`faq-knowledge-base` など）

### 4. 設定を入力
以下の設定を入力してください：

**基本設定:**
- **Name**: `faq-knowledge-base`（何でもOK）
- **Region**: `Singapore` または `Oregon`（日本に近い）

**ビルド設定:**
- **Environment**: `Node`
- **Build Command**: 
  ```
  npm install && cd client && npm install && npm run build
  ```
- **Start Command**: 
  ```
  node server/index.js
  ```

**環境変数:**
- 「Add Environment Variable」をクリック
- **Key**: `NODE_ENV`
- **Value**: `production`
- 「Add」をクリック

### 5. デプロイ開始
- 「Create Web Service」をクリック
- 5-10分待つ（ビルド中）
- 「Live」と表示されれば完了！

### 6. URLを取得
- デプロイ完了後、画面上部にURLが表示されます
- 例: `https://faq-knowledge-base.onrender.com`
- このURLを外注者に共有するだけ！

---

## トラブルシューティング

### ビルドが失敗する場合
- Build Commandが正しいか確認
- ログを確認してエラー内容を確認

### アプリが起動しない場合
- Start Commandが正しいか確認
- 環境変数 `NODE_ENV=production` が設定されているか確認

### データベースエラーが出る場合
- 最初の起動時は正常です（データベースが作成されます）
- 数秒待ってから再度アクセスしてみてください
